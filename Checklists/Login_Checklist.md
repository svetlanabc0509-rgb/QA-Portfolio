# Login Checklist (Чек-лист входу)

## UI (Інтерфейс)
- Login page is displayed correctly (Сторінка входу відображається коректно)
- Email field is visible (Поле email відображається)
- Password field is visible (Поле пароля відображається)
- Login button is visible and enabled (Кнопка входу видима та активна)
- Password is masked (Пароль прихований символами)

## Functional (Функціональність)
- User can log in with valid email and password (Користувач може увійти з валідними даними)
- Error message is shown for invalid password (Показується помилка для невірного пароля)
- Error message is shown for empty email field (Помилка для порожнього поля email)
- Error message is shown for empty password field (Помилка для порожнього поля пароля)
- Error messages disappear after исправлення даних (Помилки зникають після виправлення)

## Validation (Валідація)
- Email format is validated (Перевіряється формат email)
- Password field does not accept empty value (Поле пароля не приймає порожнє значення)

## Navigation (Навігація)
- User is redirected to dashboard after successful login (Після успішного входу користувач переходить на dashboard)
- User stays on login page after failed login (Після невдалого входу користувач залишається на сторінці входу)
