# Login Checklist (Чек-лист входу)

## UI (Інтерфейс)
- Login page is displayed correctly 
- Email field is visible 
- Password field is visible 
- Login button is visible and enabled 
- Password is masked 

## Functional (Функціональність)
- User can log in with valid email and password 
- Error message is shown for invalid password 
- Error message is shown for empty email field 
- Error message is shown for empty password field 
- Error messages disappear after исправлення даних 
## Validation (Валідація)
- Email format is validated 
- Password field does not accept empty value 

## Navigation (Навігація)
- User is redirected to dashboard after successful login 
- User stays on login page after failed login
