# QA-Portfolio

## About this repository
This repository contains my QA portfolio, created to demonstrate practical manual testing skills.  
It includes checklists, test cases, and bug reports written according to real-world QA standards.  
Bug reports are structured in a Jira-like format and reflect typical issues found during testing.

## Про репозиторій
Це QA-портфоліо, створене для демонстрації практичних навичок ручного тестування.  
Репозиторій містить чек-листи, тест-кейси та баг-репорти, оформлені за стандартами реальних проєктів.  
Баг-репорти подані у форматі, наближеному до Jira, та відображають типові дефекти.
